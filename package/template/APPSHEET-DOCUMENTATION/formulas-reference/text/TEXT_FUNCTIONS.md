# AppSheet Text Functions

Text functions manipulate and inspect strings.

## 1. Common Functions
- `CONCATENATE(text1, text2, ...)`
- `LEFT(text, number)`
- `RIGHT(text, number)`
- `MID(text, start, length)`
- `LEN(text)`
- `UPPER(text)` / `LOWER(text)`
- `CONTAINS(text, keyword)`

## 2. Examples
- `CONCATENATE([First Name], " ", [Last Name])`
- `LEFT([Order ID], 3)`
- `CONTAINS([Notes], "urgent")`
