# AppSheet List and Select Functions

Lists are a core building block for references, dropdowns, and filters.

## 1. Common Functions
- `LIST(value1, value2, ...)`
- `COUNT(list)`
- `SORT(list)`
- `UNIQUE(list)`
- `SELECT(Table[Column], condition)`

## 2. Examples
- `LIST("High", "Medium", "Low")`
- `SELECT(Orders[OrderID], [Status] = "Open")`
