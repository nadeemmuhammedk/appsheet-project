# AppSheet Date and Time Functions

Date and time functions work with Date, DateTime, and Time values.

## 1. Common Functions
- `TODAY()`
- `NOW()`
- `TIMENOW()`
- `WORKDAY(date, days)`
- `DATEDIF(start, end, unit)`

## 2. Examples
- `WORKDAY(TODAY(), 5)`
- `DATEDIF([Start Date], [End Date], "D")`
