# Table Localization

Table localization settings enable multi-language support for table and column names.

## 1. Localization Overview

**Purpose:** Display table and column names in different languages based on user locale

**Location:** Data > Tables > [Table Name] > Localization

---

## 2. Table Name Localization

**Configure display names for different languages:**

```appsheet
Default: Customers
Spanish: Clientes
French: Clients
German: Kunden
```

---

## 3. Column Name Localization

**Configure column display names per language:**

```appsheet
Column: CustomerName
Default: Customer Name
Spanish: Nombre del Cliente
French: Nom du Client
```

---

## 4. Supported Locales

AppSheet supports localization for major languages including:
- English (en)
- Spanish (es)
- French (fr)
- German (de)
- Portuguese (pt)
- Italian (it)
- Japanese (ja)
- Chinese (zh)
- And more

---

## 5. Best Practices

- Localize user-facing tables and columns
- Keep translations consistent across app
- Test with different locale settings
- Don't localize system/technical fields
- Use professional translations
